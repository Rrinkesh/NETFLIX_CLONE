# 🎬 Netflix Clone (Static UI)

A static clone of the Netflix homepage built using **HTML and CSS**.
This project replicates the modern UI design of Netflix, focusing on layout, styling, and responsiveness.

---

## 🚀 Features

* 🎥 Netflix-inspired homepage UI
* 📱 Fully responsive design
* 🎨 Clean and modern layout
* 🧭 Navigation bar with logo and menu
* 📺 Movie/TV show sections
* 🖼️ Banner section with background image
* 🔘 Hover effects on cards

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling & Layout
